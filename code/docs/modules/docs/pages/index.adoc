<h1>Documentación con Antora (code/docs)</h1>
<p>Este directorio contiene el sitio de documentación generado con Antora 3.
Incluye ingesta automática de READMEs y un modo de desarrollo offline. La búsqueda ha sido eliminada.</p>
<h2>Requisitos</h2>
<ul>
<li>Node.js 18+ (recomendado 18.x o 20.x)</li>
<li>No necesitas instalar Antora globalmente; se usa vía <code>npx</code> con las dependencias locales.</li>
</ul>
<h2>Scripts disponibles</h2>
<p>Desde <code>code/docs/</code>:</p>
<ul>
<li><code>npm run ingest</code>: Convierte los README.md a páginas AsciiDoc (módulos ROOT, karate, microservice, archetype y este <code>docs</code>).</li>
<li><code>npm run build</code> (producción con fetch): Construye con el playbook principal y <code>--fetch</code> (no necesario en entornos cerrados).</li>
<li><code>npm run build:prod</code> (producción local): Construye con <code>antora-playbook.yml</code> usando <code>ui/ui-bundle.zip</code>.</li>
<li><code>npm run build:dev</code> (desarrollo/offline): Construye con <code>antora-playbook.dev.yml</code> sin descargas (usa UI extraída).</li>
<li><code>npm run preview</code>: Construye en modo dev y sirve el sitio en <code>http://localhost:5080</code>.</li>
<li><code>npm run watch</code>: Vuelve a construir en modo dev al detectar cambios en módulos, playbooks y READMEs mapeados.</li>
<li><code>npm run preview</code>: Servidor local del sitio (5080).</li>
<li><code>npm run clean</code>: Elimina <code>build/</code> y <code>.antora-cache/</code>.</li>
<li><code>npm run ui:bundle</code>: Genera <code>ui/ui-bundle.zip</code> inyectando overrides sobre <code>ui/ui-bundle-extract/</code>.</li>
</ul>
<h2>Modo offline (recomendado en entornos corporativos)</h2>
<p>El playbook de desarrollo (<code>antora-playbook.dev.yml</code>) está configurado con <code>runtime.fetch: false</code> y usa la carpeta de UI extraída <code>./ui/ui-bundle-extract</code> para iteración rápida.</p>
<ol>
<li>Para prod determinista, genera el ZIP de UI:</li>
</ol>
<pre><code class="language-bash">cd code/docs
npm install
npm run ui:bundle
</code></pre>
<ol start="2">
<li>Ejecuta:</li>
</ol>
<pre><code class="language-bash">npm run build:dev
npm run preview
</code></pre>

<h2>Solución de problemas</h2>
<ul>
<li>“UI bundle inválido o no encontrado”: Verifica que <code>ui/ui-bundle.zip</code> existe (si usas prod) o que <code>ui/ui-bundle-extract</code> existe (si usas dev).</li>
<li>“Cambios no se reflejan”: Ejecuta <code>npm run ingest</code> y limpia la caché de Antora si fuera necesario (<code>npm run clean</code>).</li>
<li>WSL/DNS: Si tienes cortes de red, usa el modo offline; no dependemos ya de bundles remotos.</li>
</ul>
