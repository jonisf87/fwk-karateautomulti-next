= Ingesta de README
:page-component: auto-karate

El script `ingest-readmes.js` genera paginas auxiliares a partir de los README del repositorio.

== Como funciona

* Lee varios `README.md` del repositorio.
* Convierte Markdown a HTML mediante `marked`.
* Escribe paginas AsciiDoc con un bloque passthrough.
* Marca los archivos generados con `// GENERATED-BY: ingest-readmes.js`.

Si una pagina destino no contiene ese marcador, el script la preserva y no la sobreescribe.

== Cuando usarlo

* Para comparar la documentacion del sitio con lo que exponemos en README.
* Para detectar divergencias antes de reescribir README.
* Para mantener una referencia sincronizada de los README dentro del sitio.
